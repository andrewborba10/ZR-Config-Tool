Model

Model Path:

Path to player model relative to the models/ folder or other pre-defined option. (See ZR manual for more details

Model Skin Index:

The model skin index to use if the model support multiple skins. First skin is 0. (See ZR manual for more details

Initial Alpha

Initial alpha value of the player. (0 = fully invisible, 255 = fully visible)

Damaged Alpha

Alpha of a damaged player. (See 'Damage Value' for definition of a 'damaged player')

Alpha Damage Value

Amount of damage done for a player to become 'damaged'

HUD

Overlay Path:

A path relative to the 'materials' folder to an overlay to be displayed on the player's screen.

NVGs:

Yes   No

Give and turn on nightvision.

Field of Vision:

Field of vision. (90 is default)

Effects

Naplm Burn Time:

Burn duration after being hit by napalm grenade.

Player Behavior

Immunity Mode:

Sets the immunity mode. (See ZR manual for more details)

Immunity Amount:

Sets the immunity mode. (See ZR manual for more details)

Immunity Cooldown:

Sets the immunity mode. (See ZR manual for more details)

No Fall Damage:

Yes   No

Enable this to block damage from falling.

Health:

Starting HP.

Health Regeneration Interval:

The health regeneration interval, in seconds. Use 0.0 to disable.

Health Regeneration Amount:

How many health points to give per interval. Regeneration stops when back to starting HP.

Health Infect Gain:

How many health points to give the zombie each time it infects a human.

Speed:

Speed of player. NOTE: This value depends on the speed method defined in the config file. (See ZR manual for more details)

Knockback:

Knockback force multiplier.

Jump Height:

Jump height multiplier.

Jump Distance:

Jump distance multiplier.
